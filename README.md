# Snake Game

Classic Snake Game built using **HTML, CSS, and JavaScript**.  
Control the snake to eat food, grow longer, and avoid hitting walls or itself.  
Fun, interactive, and perfect for practicing JavaScript logic and DOM manipulation.

## Features
- Classic snake gameplay  
- Score tracking  
- Responsive controls  
- Works on desktop and mobile browsers
